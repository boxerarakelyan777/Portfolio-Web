## Overview

LookMate is an AI-powered fashion assistant built with Next.js, AWS, Tailwind, and Langchain. It suggests outfits based on trends, user mood, and contextual signals.

## Key Features

- Uses Langchain agents and prompt templates to generate outfit suggestions
- Incorporates feedback loops to improve personalization
- Scaled to 1,000+ users with a 25% boost in retention

## Technologies Used

- LLMs, Langchain, AWS
- Next.js, Tailwind CSS
- Custom agents + prompt tuning

## Outcome

Drove 50% increase in engagement and scaled to 1K+ users in under 2 months.
