## Overview

MemorixAI helps users learn efficiently with real-time flashcards powered by LLMs, personalization logic, and Firebase backend.

## Key Features

- Real-time flashcard generation and scoring
- Personalized study flow using AI prompts
- Stripe integration for monetization

## Technologies Used

- Next.js, React, TypeScript
- Firebase, Clerk, Stripe
- LLMs + prompt engineering

## Outcome

Boosted retention by 35% using user-behavior-driven prompts and analytics.
