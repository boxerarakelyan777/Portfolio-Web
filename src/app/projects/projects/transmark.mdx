## Overview

Built during a 48-hour hackathon, TransMark helps sales teams summarize and annotate call transcripts using LLMs and AWS.

## Key Features

- Transcript parsing, summarization, and tagging
- Modular UI for faster review
- Built in under 48 hours

## Technologies Used

- Next.js, Python, AWS, JavaScript
- LLMs + prompt engineering

## Outcome

Cut manual review time by 40% and showcased modular AI design workflows.
