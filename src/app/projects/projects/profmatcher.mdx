## Overview

ProfMatcher provides students with AI-enhanced professor insights using embeddings, vector search, and trend analysis.

## Key Features

- Pinecone + Langchain embedding pipelines
- Rating trends extracted via RAG
- Stripe + Clerk for user access control

## Technologies Used

- Next.js, Pinecone, Langchain
- Clerk, Stripe, TypeScript

## Outcome

Increased trend accuracy by 35% through tuned vector search and prompt workflows.
